# 3D时间卡片备忘录

一个具有3D效果的交互式时间卡片备忘录应用，支持创建、编辑和删除备忘录卡片。

## 功能特性

- 🕒 实时时间显示（12小时制）
- 📝 创建3D效果的备忘录卡片
- ✏️ 编辑和删除备忘录
- 🎨 毛玻璃效果背景切换
- 📱 响应式设计，支持移动设备
- 💾 本地存储，数据不会丢失

## 技术栈

- HTML5
- CSS3（3D变换、渐变、动画）
- JavaScript（ES6）
- LocalStorage（数据持久化）

## 使用方法

1. 在标题输入框中输入备忘录标题
2. 在内容输入框中输入备忘录内容
3. 点击"创建备忘录"按钮
4. 卡片会自动添加到列表中
5. 可以点击卡片上的按钮进行编辑、清空或删除操作

## 设置选项

- 背景效果切换：可以开启/关闭毛玻璃背景效果

## 部署说明

本项目已通过GitHub Pages部署，可以直接访问。

## 本地运行

直接打开`index.html`文件即可在浏览器中运行。
